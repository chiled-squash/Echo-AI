# Echo-AI
This is a Chrome extension. When you enter a question on any AI platform, the Echo-AI extension automatically sends the same content to other open AI pages, helping you quickly compare responses from different models and improve efficiency and decision-making quality.这是一个Chrome插件，当你在任意一个 AI 平台输入问题时，Echo-AI插件会自动将相同内容发送到其他已开启的 AI 页面，帮助你快速对比不同模型的回答结果。
